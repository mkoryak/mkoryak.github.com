Working with datatables

jquery.floatThead works out of the box with datatables or any other table framework.
Be sure to init the framework first, and then the plugin.

Code

$(document).ready(function () {
     var $table = $('#demo0');
     $table.dataTable({
         "bInfo": false,
         "iDisplayLength": 100,
         "bLengthChange": false
     });

    $table.floatThead({
        scrollingTop: 41 //i need this because of my floating header
    });
});

Inner Scrolling Example

Code

$(document).ready(function () {
    var $table2 = $('#demo1');
    $table2.dataTable({
        "bInfo":false,
        "iDisplayLength":1000,
        "bLengthChange":false
    });

    $table2.floatThead({
        scrollingTop:41,
        scrollContainer: function($table){
            var $wrapper = $table.closest('.dataTables_wrapper');
            $wrapper.css({'overflow': 'auto', 'height': '400px'});
            return $wrapper;
        }
    });
});

Using Datatable's FixedHeader plugin

Code

$(document).ready(function () {
    var oTable = $('#demo2').dataTable();
    new FixedHeader( oTable );
});
This example shows the FixedHeader plugin written by datatable's author. I put it here to compare it with my plugin. FixedHeader does not support inner scrolling.
